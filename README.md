## Установка
Скопируйте репозиторий проекта командой:
```sh
$ git clone https://github.com/AbShamilDev/twitchat.git
```
Установите все зависимости командой:
```sh
$ npm i 
```
Запустите проект командой:
```sh
$ npm start 
```
